# 🏏 IPL Analytics Executive Dashboard

## 📌 Project Overview
This project provides a comprehensive data analysis of the Indian Premier League (IPL). It aims to uncover strategic insights regarding team dominance, high-impact players, and match conditions by analyzing historical match data. The insights are synthesized into an executive report designed to drive data-informed strategies and tactical decisions.

## 📊 Dataset Summary
The analysis is built on a robust historical dataset encompassing:
* **Total Matches:** 950
* **Total Teams:** 18
* **Total Players:** 605

## 🎯 Key Business Questions & Insights
The notebook and accompanying dashboard address several core strategic questions to evaluate performance and trends:

* **Which teams dominate the IPL?**
  * *Methodology:* Analyzed total matches, win ratios, and title counts.
  * *Insight:* **Mumbai** emerges as a consistently dominant and strong force in the tournament.
* **Which players drive performance?**
  * *Methodology:* Evaluated high-impact players through top wicket-takers and Player of the Match awards. 
  * *Insight:* Key drivers of performance include DJ Bravo, AB de Villiers, and Virat Kohli.
* **Does the toss influence match results?**
  * *Methodology:* Compared toss winners with final match outcomes to refine match strategy.
  * *Insight:* The toss has a **47% impact** on the final result, highlighting its strategic importance.
* **What are the seasonal trends?**
  * *Methodology:* Tracked matches season-by-season.
  * *Insight:* There is a steadily increasing trend in tournament growth, indicating rising fan engagement over time.
